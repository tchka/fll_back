Учебный проект - фриланс биржа.
В проекте реализована регистрация пользователя с выбором исполнитель/заказчик и система создания и выполнения заявок на работы.
Выполнение заявок происходит через изменение полей заявок и их статуса в лк пользователей. В зависимости от типа кабинета (исполнитель/заказчик), авторства заякви и статуса пользователю доступны для редатирования определенные поля.
